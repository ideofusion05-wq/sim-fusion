# SimFusion

Marketing website for **SimFusion** — a research & innovation studio building AI, XR and modern software.

A single-page, animated static site (no build step) inspired by a dark, high-contrast studio aesthetic (dark theme + yellow accent, bold display type, scroll animations, particle-network hero).

## Services showcased

1. Research & Innovation
2. AI Technologies
3. Conferences (research + software)
4. VR / AR / MR Application Development
5. Medical & Education Software
6. Web, Android & iOS Apps
7. Game Development
8. UI / UX Design
9. Digital Marketing

## Structure

```
index.html      # page markup / all sections
styles.css      # theme, layout, animations
script.js       # nav, scroll reveal, hero canvas, counters, contact form
assets/         # images (hero, service & feature photos)
```

## Run locally

Just open `index.html` in a browser, or serve the folder for best results:

```bash
python3 -m http.server 8777
# then visit http://localhost:8777
```
